启动命令：
npm run start || node main.js

项目输入路径：
config.js/base

pipe：

      vue  -- template -- plugins

           -- script   -- plugins

      js   -- plugins

文件节点:  

      value :  文件内字符串

      type  :  file/dir
          
      path  :  绝对路径
          
      name  :  文件全名

      log   :  日志

使用或调试需自行打开pipe函数
由于转换优先级策略，请勿随意调换管道顺序
